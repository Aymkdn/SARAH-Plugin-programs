Le module programs permet, par défaut, de lancer XBMC :


SARAH, démarre XBMC


Par défaut le path vers XBMC est C:\\Program Files (x86)\\XBMC\\XBMC.exe -- à modifier dans le fichier programs.js.

Ce plugin est mis à disposition comme exemple et peut être entièrement personnalisé, en particulier parce que le répertoire bin/ contient plusieurs programmes pour arrêter un processus, ou verrouiller l'ordinateur et autres.
